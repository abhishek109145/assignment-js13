    Ques: Write the code to access element which is having id as "text".
    
 Ans: const elem = document.getElementById('text');
    

    Ques: Write the code to access element which is having tag as "h1".
    
 Ans: const elem = document.getElementsByTagName('h1');
    

    Ques: Write the code to access element which is having class as "box".
   
 Ans: const elem = document.getElementsByClassName('box')
